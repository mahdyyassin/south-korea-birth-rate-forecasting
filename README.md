Overview:
Forecasting monthly live births in South Korea using time series techniques.

Dataset:
Monthly births
1997-2024
336 observations

Methods:
Classical Decomposition
ADF Test
KPSS Test
Differencing
SARIMA
Box-Jenkins Methodology

Final Model:
ARIMA(0,1,4)(0,1,1)[12]

Results:
Generated forecasts through 2029.

The model predicts a continued gradual decline in birth rates.
